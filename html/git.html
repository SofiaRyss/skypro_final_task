Git

  • git init - расшифровывается как "Создание нового репозитория Git в пустой директории или превращение существующей директории в новый репозиторий".
  • git status - расшифровывается как "Просмотр списка измененнных файлов".
  • git add - расшифровывается как "Добавление в отслеживаемые файлы".
  • git commit - расшифровывается как "Сохранение изменений в Git".
  • git push - расшифровывается как "Выгрузка сделанных изменений в удаленный репозиторий,например,в Git".
  • git pull - расшифровывается как"Получение изменений из удаленного репозитория на локальную машину"./
